MendoAPI

Una API pública con datos abiertos y organizados de Mendoza, Argentina.

Explorar endpoints Ver en GitHub

¿Para qué sirve MendoAPI?

  • 📚 Educación: ideal para proyectos escolares o ejercicios de programación
  • 🧭 Turismo: datos base para apps de turismo y cultura
  • 📊 Visualizaciones: alimentar gráficos con población, superficie y regiones
  • 🌐 Gobierno abierto: reutilización de datos sin restricciones

Endpoints disponibles

  • GET /api/departamentos – Lista completa de departamentos
  • GET /api/departamentos/:id – Datos de un departamento específico
  • GET /api/regiones – Lista de regiones reconocidas
  • GET /api/regiones/:region – Departamentos dentro de una región

🔗 Base URL: https://mendoapi.up.railway.app
